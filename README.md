# Basic-Webserver-with-UPnP

Really fast and dirty webserver with UPnP to open port 80. 

Basic handling.

Browse to /end to close the port again.

#To Run:

go get github.com/NebulousLabs/go-upnp

go run UPnPWebserver.go
